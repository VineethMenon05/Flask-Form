# 📝 Flask Form Application

A simple and efficient web application built using Flask that allows users to submit forms and process input data dynamically.

---

## 🚀 Features
- 🧾 User form submission  
- ✅ Input validation  
- 📊 Data handling using Flask backend  
- 🎨 Clean UI with HTML & CSS  
- 🔄 Dynamic rendering using Jinja2 templates  

---

## 🛠 Tech Stack
- **Backend:** Flask (Python)  
- **Frontend:** HTML, CSS  
- **Templating:** Jinja2
